Add a custom text box to the blog post footer to encourage reader interaction.

== Description ==
This plugin adds a highlighted text box to the bottom of every blog post.

== Installation ==

If you are using the Plugins tab on your wordpress blog to view and install plugins, simply click 'Install Now' and it will install the plugin automatically for you. All you need to do then is "Activate" it.

Here is how to get this plugin working if you downloaded it to your desktop.


1. Upload `post-footer-box.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it!

If you need to change or edit anything on the plugin, you can find it under Settings -> Post Footer.
